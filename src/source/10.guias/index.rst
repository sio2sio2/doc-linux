Guías para ciclos formativos
============================
Se incluyen dentro del documento guías para el desarrollo de algunos
módulos de los ciclos formativos de |ASIR| y |SMR|, cuyos contenidos en gran medida
pueden desarrollarse tomando partes del manual. Dado que éste es eminentemente
práctico, mientras que las guías pretenden enumerar los contenidos mínimos de
sus respectivos módulos, éstas contendrán:

- Contenidos de caracter teórico.
- Índices para desarrollar las partes prácticas sin aplicación en *Linux* (las
  referentes a *Windows*, p.e.)
- Enlaces a las partes pertinentes del manual.
- Ejercicios complementarios.

.. note:: Las guías son todas las que están, pero no están todas las que son;
   esto es, pueden faltar las guías de módulos que sean apropiados sean el
   criterio enunciado bajo el epígrafe.
