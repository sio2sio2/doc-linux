5.8. Instalación del servidor¶

El tema expone cómo instalar una debian sobre una máquina virtual con el propósito de montar el servidor en ella antes de trasladarlo a la máquina real.

Se discutirá cómo particionar el disco y llevar a cabo este particionado mediante volúmenes lógicos; y cómo crear un RAID 1 para asegurar mejor los datos frente al fallo fortuito de un disco.

Ambas tareas se realizarán con el propio instalador de debian, pero en los enlaces proprcionados se discute en profundidad sobre ambos aspectos y se puede consultar cómo realizar esta tarea a mano antes de comenzar la instalación, de manera que en esta nos limitemos a indicar qué parte del árbol de directorios se corresponde con qué partición (o, por mejor decir, con qué volumen lógico).
