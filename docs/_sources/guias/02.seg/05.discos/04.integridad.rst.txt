Recuperación
============
Cuando se ha producido una pérdida, podemos intentar recuperar los datos
originales mediante dos estrategias distinta\ [#]_:

+ Mediante técnicas de recuperación, que se analizarán en :ref:`la unidad
  dedicada a los sistemas de escritorio <seg-desktop>`.

+ Si fuimos previsores, podemos restaurar a un estado anterior que establecimos
  como referencia. A ello dedicaremos el resto del epígrafe.
