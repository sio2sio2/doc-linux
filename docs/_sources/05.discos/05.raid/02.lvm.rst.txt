.. _lvmraid:

|LVM|
=====
Desde su versión 2, |LVM| permite la creación y gestión de dispositivos |RAID|
como si de volúmenes lógicos se tratara. Es pues una buena alternativa si
nuestra intención es crear volúmenes lógicos con redundancia, en vez de crear el
|RAID| y crear después sobre el dispositivo virtual resultante un grupo de
volúmenes. La gestión de volúmenes lógicos redundantes tiene su propia página de
manual: :manpage:`lvmraid(7)`.
