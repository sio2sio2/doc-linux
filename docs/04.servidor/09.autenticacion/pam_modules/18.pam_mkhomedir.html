4.4.4.18. pam_mkhomedir¶

Descripción

Crea el directorio de usuario si no existe aún.

Grupos

Para session.

Argumentos

silent

No informa.

umask=MASCARA

Define la máscara con la que se creará el directorio personal. La predeterminada es 0022.

skell=PATH

Indica que plantilla se usará para crearlo. El valor predeterminado es /etc/skell
