Ejercicios sobre rutas¶

Nota

Observe las siguientes reglas:

  1. No cambie de usuario, si no se se especifica expresamente o es necesario para realizar la operación requerida.

  2. Escriba siempre la orden anteponiendo como prompt un almohadilla (#), si el usuario es root, o un dólar ($), si es cualquier otro.

  3. No cambie de directorio, si no se le ordena.

Usando el usuario del sistema sin privilegios:

  1. Vaya al directorio general de configuraciones.

  2. Sin salir de ese directorio, consulte el contenido del directorio / usando una ruta relativa.

  3. Usando una ruta relativa, pase al directorio home dentro del cual se encuentran los datos personales de los usuarios.

  4. Consulte el contenido del subdirectorio personal .config. ¿Qué ruta es más corta? ¿La relativa o la absoluta? Nota: El directorio puede no existir si el sistema no se ha utilizado mucho.

  5. Compruebe cuál es el directorio actual de trabajo.

  6. Pásase al directorio local dentro de usr.

  7. Suponiendo que el administrador del sistema respete el estándar, ¿hay algún ejecutable en el sistema que haya instalado él manualmente sin usar ningún gestor de paquetes?

  8. Con ruta relativa, volver al directorio personal.

  9. Listar el contenido del directorio raíz usando ruta relativa.

  10. Suponiendo que los directorios personales de usuario estén en su ubicación habitual, obtenga la lista de tales directorios a fin de estimar cuántos usuarios físicos tiene definidos el sistema. Nota: Esto sólo es una aproximación hasta que aprendamos cómo consultar usuarios.
