6. Web¶

La configuración de un servidor web puede complicarse mucho, así que nos limitaremos a aprender la configuración básica del servidor nginx

6.1. Introducción¶

Debe comprender la introducción del manual y la descripción del protocolo. De esta última es conveniente hacer un repaso a todo lo expuesto, ya que contribuye a entender cómo funciona el servicio.

6.2. Configuración básica¶

  • Principios de configuración.

  • Definición de dominios virtuales.

  • Bloque location.

6.3. Configuración adicional¶

  • Control de accesos.

  • Conexión segura. Es fundamental obtener un certificado acreditado, aunque entrañe más dificultad.

  • Reescritura de direcciones y sentencial condicional, esta última sobre todo para propiciar reescrituras.

6.4. Contenido dinámico¶

  • Generación de páginas dinámicas con PHP, sin llegar a entrar en las disquisiciones sobre cache.

  • Instalación de una aplicación sencilla como DMS o phpBB.
