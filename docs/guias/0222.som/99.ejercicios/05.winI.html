Ejercicios sobre configuración de Windows 10¶

  1. Una pantalla de acceso a Windows 10, muestra este aspecto:

    Se pide:

    1. Identificar las distintas partes que componen la plantlla.

    2. Para cada una de ellas:

      • Explicar para qué sirve.

      • Incluir una captura si al pinchar sobre ella se muestra información adicional.

      • Deducir cuál es el estado de nuestro propio sistema según la información que recibamos.

  2. Lleve a cabo las siguientes operaciones, haciendo una captura o varias capturas que confirmen que es capaz de hacerlas:

    1. Salga de la sesión.

    2. Apague el equipo (pero no desde la pantalla de acceso).

    3. Utilice «Cortana» para abrir la Configuración del Sistema.

    4. Acceda a la aplicación de «Administrador del Equipo».

    5. Acceda a la sección de «Herramientas administrativas».

    6. Acceda a la sección de «Administrador de tareas».

    7. Acceda a la configuración del escritorio para:

      • Sustituya la imagen de fondo por un color sólido.

      • Indique dónde puede cambiar la resolución de pantalla.

      • Deshabilitar el apagado de temporización.

    8. Modifique el menú de Inicio para que aparezcan las aplicaciones más usadas en vez de las más recientemente agregadas.

    9. En los iconos de escritorio:

      • Elimina el de «Papelera».

      • Añada el de «Red».

  3. Realice las siguientes operaciones:

    1. Llegue al directorio personal del usuario partiendo del disco C:.

    2. Llegue al directorio donde se guardan las aplicaciones al ser instaladas, partiendo de C:.

    3. Cree un directorio llamado «MisCosas» en D: y dentro de él dos subdirectorios llamados «Documentos» y «Descargas».

    4. Modifique las ubicaciones de «Documentos» y «Descargas» para que sean los directorios anteriormente creados.

    5. Haga que se muestren siempre las extensiones de los archivos, incluso aunque sean de tipos conocidos.

    6. Compruebe que al crear un fichero con el Block de notas y guardarlo en «Documentos», se almacena en D:, no en C:.

    7. Compruebe que al descargar una imagen de Internet se almacena en D:.

    8. Indicar cuál es la ocupación de C: y D:

  4. Lleve a cabo las siguientes acciones:

    1. Busque en internet la aplicación «Visual Studio Code» (¡ojo!, no «Visual Studio») e instálela en el sistema.

    2. Asocie los archivos de extensión RST para que se abran con esta aplicación.

    3. Cree un fichero vacío con esta extensión y compruebe que se abre automáticamente con la ella.

    4. Desinstálela.

    5. Acceda a la configuración de red y consulte cuál es. Actívela y desactívela.

    6. Consulte la información de su sistema (procesador, memoria, versión del sistema operativo, etc.)

    7. Compruebe que su sistema dispone de cortafuegos y de que éste está activo (Windows Defender).

  5. Configure el sistema para que:

    • Se apague de lunes a viernes a las 15:00 y a las 23:00 de la noche.

    • Al iniciar la sesión del usuario, se abra automáticamente el explorador de archivos.
