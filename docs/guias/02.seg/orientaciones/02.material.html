Material adicional¶

Para la realización de algunas de las prácticas del curso se requerirá:

  1. Una máquina virtual con la última distribución de Debian, ya que varias de los ejercicios requieren la práctica de los conceptos teóricos dentro de un entorno Linux.

  2. Puntualmente, un sistema Windows 10, preferentemente virtualizado para evitar trastocar fatalmente un sistema de trabajo.

  3. Algún SAI, aunque como puede emularse su comportamiento, no es absolutamente indispensable.

  4. Un servidor virtual en internet con IP pública1 que servirá para:

    • Obtener certificados de servidor.

    • Crear un servidor VPN.

Notas al pie

1

El programa Azure Dev Tools for Education, por ejemplo, permite la creación de servidores VPS a alumnos y profesores
