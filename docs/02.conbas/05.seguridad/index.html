2.3. Seguridad del sistema¶

Bajo este epígrafe trataremos el modo básico que existe en Linux para asegurar el sistema y los datos, esto es, la gestión de usuarios y permisos. Son conceptos íntimamente ligados ya que la exustencia de distintos usuarios se justifica por la necesidad de asociarles distintos permisos sobre los recursos del sistema y los datos que se almacenan.
