Linuxnomicón¶

El documento tiene por propósito desarrollar los temas necesarios para hacer de un sistema Linux basado en Debian un servidor que gestione las redes locales y proporcione servicios a internet. El tema es muy amplio, pero se ha procurado incluir todos los aspectos relevantes, desde los más básicos a otros de cierta enjundia y profundidad. Esta es la razón por la que el texto carece de apartados dedicados a la configuración de entornos gráficos.

Guías para ciclos formativos¶

Se incluyen dentro del documento guías para el desarrollo de algunos módulos de los ciclos formativos de ASIR y SMR, cuyos contenidos en gran medida pueden desarrollarse tomando partes del manual. Dado que éste es eminentemente práctico, mientras que las guías pretenden enumerar los contenidos mínimos de sus respectivos módulos, éstas contendrán:

  • Contenidos de caracter teórico.

  • Relación con resultados de aprendizaje y criterios de evaluación

  • Índices para desarrollar las partes prácticas sin aplicación en Linux (las referentes a Windows, p.e.)

  • Ejercicios complementarios.

Nota

Las guías son todas las que están, pero no están todas las que son; esto es, pueden faltar las guías de módulos que sean apropiados según el criterio enunciado bajo el epígrafe.
