7.2.1.5. Cuerpo¶

Tanto la petición como la respuesta pueden contener un cuerpo separado de la cabecera mediante una línea en blanco. Sobre el cuerpo no hay demasiado que explicar:

  • En las peticiones mediante el método POST, el cuerpo contiene datos que se envían al servidor para que alimentar la entrada de una aplicación web empotrada.

  • En las respuestas, el cuerpo contiene el recurso solicitado (p.e. una página web). Para que el cliente sepa de qué tipo es (si una imagen, si texto en claro, si un documento HTML, etc), la cabecera contiene una etiqueta Content-Type con el tipo MIME asociado al recurso. Por ejemplo, al devolverse una página HTML la cabecerá contiene una línea así:

    Content-Type: text/html
