7.2.2.3. Recetas¶

Se exponen aquí algunas recetas para la configuración de aspectos particulares de nginx y la instalación de aplicaciones web habituales.

Advertencia

Para la confección de estas recetas, cuando la configuración exige tráfico seguro, se ha usado la configuración con certificado autofirmado, aunque no es la recomendable. Se obra así, porque para realizar pruebas de instalación en máquinas virtuales, es lo más cómodo. Cambiar la configuración para usar claves de Let’s Encrypt es, no obstante, trivial.
